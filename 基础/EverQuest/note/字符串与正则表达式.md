

#### 字符串与正则表达式

##### 常用操作

-  拼接

  ```python
  str = str1 + str2
  ```

- 长度

  ```python
  len('人生苦短，我用python!')  # 14
  len('人生苦短，我用python!'.encode())  # utf-8 28
  len('人生苦短，我用python!'.encode('gbk'))  # gbk 21
  ```

- 截取

  ```python
  string[start : end : step]
  ```

- 分割

  ```python
  str.split(sep, maxsplit)
  # sep:指定分隔符，默认空白符
  # maxsplit:可选参数，用于指定分割的次数
  ```

- 检索

  ```python
  # 检索指定字符串在另一个字符串中出现的次数
  str.count(sub[, start[, end]])
  # 检索是否包含指定的子字符串，返回-1或者索引
  str.find(sub[, start[, end]])
  # 检索是否包含指定的子字符串
  str.index(sub[, start[, end]])
  # 检索字符串是否以指定字符串开头
  str.startwith(prefix[, start[, end]])
  # 检索字符串是否以指定字符串结尾
  str.endswith(suffix[, start[, end]])
  ```

- 转换

  ```
  str.lower()
  str.upper()
  ```

- 去除

  ```python
  # 去除字符串左右两侧的空格和特殊字符
  str.strip([chars])
  str.lstrip([chars])
  str.rstrip([chars])
  ```

- 格式化

  ```
  # 使用’%‘操作符
  '%[-][+][0][m][.n]格式化字符'%exp
  # 使用字符串对象的format()方法
  str.format(args)
  {[index][:[[fill]align][sign][#][width][.precision][type]]}
  ```
